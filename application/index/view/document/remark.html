Game Brain 广告参数基础字段

在Game Brain创建应用后,以下基础字段为广告参数所必须字段，再您添加参数时，请务必添加

URL

http://console.gamebrain.io/advertise/advlist?appid=xx

支持格式

JSON

HTTP请求方式

GET/POST

基础字段(*)
参数 必选 类型 默认 说明
interadsinteral ture string 30 新用户前多少秒没有插屏广告
noadtime ture string 120 新用户前多少秒无广告，单位秒
interadscount ture string 0 插屏局数控制，几局出插屏广告，默认是0
noadday ture string 30 新用户前几天无插屏，单位天，默认值是0
interChangeToVideoCount ture string 0 插屏转视频的次数，默认是0，不转；具体数值就代表转换几次
interChangeToVideoIntervalTime ture string 3 插屏转视频，间隔视频的时间，默认是3分钟
AuditAds ture string 0 审核期间，广告总开关，默认是0，默认总开关关；1是开
NetWorkDetectionCanClose ture string 0 网络检测期间，默认是0，显示'稍后再说'文案，点击可以关闭；若设置成1，显示“刷新”文案，必须设置才可关闭
返回格式
	  
	    {
		   "int": [
					{
					  "name": "am1",
					  "val": "ca-app-pub-6491984961722312/7205269193"
					}
			  ],
		   "rew": [],
		   "nat": [],
		   "ban": [],
		   "conf": {
			"InterTimeOne": "60",
		   },
		  "country": ""
        }
内购请求

用于统计每个产品的内购收益情况

URL

http://console.gamebrain.io/purchase/index

支持格式

JSON

HTTP请求方式

GET/POST

基础字段(*)
参数 必选 类型 说明
appid ture string 应用ID
income ture string 用户每次购买的金额
orderid ture string 商品购买交易ID
country ture string 用户所属国家简写标志 CN
返回格式
	  
	    {		  
		  "code":200
        }
